### 基本介紹
對於程式開發，Git的功能絕對不能不知道，Git可以幫助使用者紀錄文字檔案撰寫歷史，很常使用在程式設計使用上，可以讓個人或多人了解檔案撰寫過程，達到追蹤及備份等功能，甚至可以透過雲端裝置，例如：GitHub，讓使用者在多個裝置中無縫接續當前的檔案繼續編輯。這邊會說到**文字檔案**編輯想要強調並不是只有程式設計才可以使用Git，只要是文字檔案，其實都可以透過Git來幫助編輯修改追蹤。

### 常用指令
- git status
- git log
- git add
- git commit
- git push
